Python Brazil [11]


### Set up & Install dependencies

```
$ git clone git@github.com:designa/pythonbrasil11.git
$ cd pythonbrasil11
$ gem install jekyll
```

### Server

```
$ jekyll serve
```

### Gulp

```
# To install gulp deps
$ npm install

# If you need to rebuild javascript
$ gulp
```

### Using a browser, go to:

```
http://localhost:4000
```
